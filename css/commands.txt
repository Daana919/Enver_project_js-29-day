ls(list) - выводит список содержимого в папке (вкл папки и файлы)
cd(change darectory) - переход между папками
pwd(path working directory) - абсолютный путь к папке в которой вы находитесь
ls -a - скрытые папки и файлы
rm -rf (.название папки) - принудительное удаление папки
git init - инициализация git репозитория 
git remote add origin <SSH-url>- привязка локадбного репозитория к удаленнному 
git remote -v - посмотреть к какому репозиторию привязан локальный репозитори
